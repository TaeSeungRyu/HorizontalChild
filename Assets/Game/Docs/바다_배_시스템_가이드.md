# 바다 · 배 시스템 가이드

바다(파도)와 배(부력·이동·항적) 관련 스크립트 사용법입니다.
스크립트 위치: `Assets/Game/Scripts/`

| 스크립트 | 역할 | 붙이는 곳 |
|----------|------|-----------|
| `OceanWaves.cs` | 출렁이는 바다 생성 | 빈 GameObject(바다) |
| `FloatOnWaves.cs` | 배가 파도에 떠서 흔들림(부력) | 배 |
| `ShipMover.cs` | 배 조종(W/S 전진, A/D 회전) | 배 |
| `ShipWake.cs` | 배 항적/물보라 | 배 |

> 셰이더를 직접 안 만지도록 설계되어 핫핑크(마젠타) 문제가 거의 없습니다.
> 그래도 핫핑크가 나오면 머티리얼 셰이더를 `Universal Render Pipeline/Lit`(파티클은 `.../Particles/Unlit`)로 바꾸세요.

---

## 1. 바다 — OceanWaves

빈 GameObject에 추가하면 격자 메쉬를 스스로 만들어 매 프레임 출렁입니다.

**설정:**
- **Size / Resolution** — 바다 크기 / 격자 촘촘함(높을수록 부드럽지만 무거움)
- **Wave Height** — 파도 높이 (흔들림 세기). 0이면 잔잔(=부력 효과도 사라짐)
- **Wave Speed** — 출렁이는 속도
- **Wave Scale** — 작을수록 크고 완만한 너울, 클수록 잔물결
- **Water Level** — 수면 높이(Y). 육지가 물을 뚫으면 이 값을 내려 해결
- **Follow Camera** — 켜면 카메라를 따라다녀 맵 전체를 덮음(맵 크기 무관)
- **Sea Color** — 바다색

**맵 전체 덮기:** Follow Camera 체크 → Size는 시야를 덮을 만큼(예: 200~400), 화면 끝에 경계가 안 보일 때까지 키우기.

**물결은 월드 좌표 기준** → 타일을 여러 개 깔아도 이음새가 없음.

---

## 2. 배 부력 — FloatOnWaves

배가 파도 표면에 떠서 오르내리고, 파도 경사에 따라 기웁니다. 방향(yaw)은 안 건드림.

**설정:**
- **Ocean** — 비우면 자동으로 씬의 OceanWaves를 찾음
- **Float Offset** — 수면 기준 높낮이(- 값이면 살짝 잠김)
- **Sample Length / Width** — 앞뒤/좌우 기울기 감지 거리. 배 길이·폭의 절반쯤. 클수록 완만, 작을수록 출렁임 강조. 배를 스케일했으면 같이 키우기
- **Move / Rotate Smooth** — 따라가는 부드러움

**프리팹에 붙여도 됨** — Ocean 칸은 비워두면 실행 시 자동 연결(프리팹은 씬 참조 불가).

**효과가 약하면:** OceanWaves의 Wave Height를 올리기(잠깐 2~3으로 테스트). 너무 약하면 안 보일 뿐 정상 동작.

---

## 3. 배 조종 — ShipMover

이동 스크립트가 따로 없을 때 사용.
- **W/S (↑/↓)** = 바라보는 방향으로 전진/후진
- **A/D (←/→)** = 좌우 방향 전환
- **Move Speed / Turn Speed** — 속도 조절

위치(X/Z)와 방향(Yaw)만 제어하고, 높이·기울기는 FloatOnWaves가 맡아 둘이 충돌 없이 동작.

> 배가 방향과 다르게 미끄러지면: FloatOnWaves가 방향을 고정하지 않는지(최신 버전), Rigidbody가 있으면 Is Kinematic 체크.

---

## 4. 배 항적/물보라 — ShipWake

배가 움직이면 뒤에 하얀 거품 자국 + 뱃머리 물보라. 파티클·머티리얼·거품 텍스처를 코드로 자동 생성(핫핑크 없음).

**설정:**
- **Stern Offset** — 항적이 나오는 뒤쪽 위치(-Z, 음수). 배 뒤 끝에 맞추기
- **Bow Offset** — 뱃머리 물보라 위치(+Z)
- **Wake Width** — 항적 폭(배 폭쯤)
- **Water Y** — 거품 생기는 높이(배 바닥 0 기준이라 0.1~0.2). 물에 묻히면 올리기
- **Foam Size / Density** — 거품 크기 / 진하기
- **Foam Color** — 거품 색

움직인 **거리에 비례**해 거품이 생겨서, 멈추면 자동으로 안 나옴. 배를 스케일했으면 Offset·Width도 같이 키우기.

---

## 권장 조립 순서

1. 빈 GameObject에 **OceanWaves** → 바다 만들기 (Follow Camera로 맵 덮기, Water Level로 육지 위로)
2. 배(프리팹)에 **FloatOnWaves** + **ShipMover** + **ShipWake** 추가
3. Play → 바다가 출렁이고, 배가 떠서 흔들리며, W/A/S/D로 조종, 움직이면 항적이 생김

## 자주 겪는 문제

- **핫핑크** → 셰이더를 URP용으로 변경
- **부력 효과 없음** → Wave Height가 0/너무 작음 → 올리기 (육지 뚫림은 Water Level로 잡기)
- **배가 미끄러짐** → FloatOnWaves가 방향 고정 안 하는 최신 버전인지 / Rigidbody Is Kinematic
- **항적이 물에 묻힘** → ShipWake의 Water Y 올리기
