# 산/언덕 배치 가이드

지정한 자리에만 산을 올리는 방법입니다. **지도·바다를 놓던 것과 똑같이** 모델을 씬에 끌어다 놓으면 됩니다.

- 모델: `Assets/Game/Art/Models/Nature/`
  - `Mountain_Big` (눈 덮인 큰 산), `Mountain_Peak` (바위 봉우리), `Hill_Round` (둥근 언덕)
- (선택) 스크립트: `Assets/Game/Scripts/MountainPlacer.cs`

---

## 방법 1 — 손으로 놓기 (가장 간단, 추천)

지도·바다 놓던 방식과 동일합니다.

1. Project 창에서 `Mountain_Big`(또는 다른 산) 찾기
2. **Scene 뷰 / Hierarchy로 드래그**
3. **이동 툴(W)**로 원하는 자리로 옮기기
4. **크기 조절(R 또는 Transform Scale)** — 실측 지도면 작게 보이니 **많이 키우기**
5. **높이(Y)** 를 맞춰 밑동이 땅에 닿게. 살짝 아래로 내려 박으면 틈이 안 생김
6. 자리마다 반복 (복제는 **Ctrl+D**)

> 소수의 산을 특정 위치에 둘 때 가장 확실하고 통제가 쉬움.

---

## 방법 2 — 마커로 여러 개 관리 (MountainPlacer)

산이 많거나 지면 높이에 자동으로 맞추고 싶을 때.

1. 빈 GameObject 생성 → `Add Component` → **Mountain Placer**
2. 그 아래에 **빈 자식(Empty)** 을 만들어 산 둘 자리로 이동 (개수만큼, 마커엔 노란 구 표시)
3. **Mountain Prefabs**에 산/언덕 프리팹 넣기 (여러 개면 무작위 선택)
4. 컴포넌트 우클릭(⋮) → **Place Mountains** (지우려면 **Clear**)

설정값:
- **Snap To Ground** — 마커 X/Z에서 지면 높이를 찾아 앉힘 (지형에 Collider 필요)
- **Sink Into** — 바닥에 살짝 박아 틈 방지
- **Scale Range** — 크기 무작위 (큰 맵이면 많이 키우기)
- **Random Yaw** — 무작위 회전

---

## 공통 팁

- **색이 핫핑크(마젠타)면** → 머티리얼 셰이더를 `Universal Render Pipeline/Lit`로 변경
- **Collider** — 손으로 놓을 땐 불필요(보이기만 하면 됨). 플레이어가 산에 부딪히게 하려면 산에 **Mesh Collider** 추가. (MountainPlacer의 Snap은 지형 Collider 필요)
- **크기** — 실측 지도는 크므로 산도 크게. 한 번 놓고 맵과 비교하며 Scale 조절
- **각져 보이면** — 더 자연스러운 모양으로 다듬는 것도 가능(요청)

---

## 참고 — "산 모양 얹기" vs "지형 자체 변형"

- **(A) 산 모양 얹기** — 위 방법. 지형 위에 산 메쉬가 올라앉음. 쉽고 어떤 지형에서도 됨. **대부분 이걸로 충분.**
- **(B) 지형 바닥 자체가 솟아오르게(변형)** — 땅 표면이 매끄럽게 부풀어 산이 됨. 더 복잡하며 지형 종류에 따라 다름:
  - Unity **Terrain** → 높이맵을 그 자리만 올리기 (가능)
  - **메쉬 지형**(지도 데이터) → 정점 변형 필요(작업 많음)

(B)가 필요하면 지형이 Terrain인지 메쉬인지 알려주세요.
