# 자연물(나무·바위·풀) 배치 가이드

맵을 채우기 위한 **프리팹 흩뿌리기(NatureScatter)** 사용법입니다.
직접 만든 모델이든, 받아둔 에셋이든 똑같이 적용됩니다.

---

## 0. 들어가는 파일

- 자연 모델: `Assets/Game/Art/Models/Nature/`
  - `Tree_Pine` (소나무), `Tree_Round` (둥근나무), `Rock` (바위), `Bush` (덤불)
- 스크립트: `Assets/Game/Scripts/NatureScatter.cs`

> 모델은 전부 초저폴리(20~60 tri)라 수백 개를 뿌려도 가볍습니다.

---

## 1. 모델을 프리팹으로 만들기

1. `Nature` 폴더의 FBX(예: `Tree_Pine`)를 **씬(Hierarchy)으로 한 번 드래그**
2. 그 오브젝트를 다시 **Project 창의 폴더(예: `Assets/Game/Prefabs`)로 드래그** → 프리팹 생성
3. 씬에 올렸던 원본은 지워도 됨
4. 나무·바위·덤불 각각 반복

> 색이 핫핑크로 나오면 머티리얼 셰이더를 **URP/Lit**로 바꿔주세요
> (Project에서 머티리얼 선택 → Inspector 상단 Shader → `Universal Render Pipeline/Lit`).

---

## 2. 기존(받아둔) 프리팹 사용하기

**그대로 됩니다.** NatureScatter는 어떤 프리팹이든 받아서 뿌립니다.

- Project 창에 있는 **프리팹**(파란 아이콘)을 Prefabs 칸에 드래그
- 제가 만든 로우폴리 모델과 **섞어서** 같이 넣어도 됨
- 주의할 점
  - **피벗(중심점)**: 프리팹 피벗이 바닥에 있어야 땅에 딱 붙음.
    가운데에 있으면 절반이 묻히거나 뜸 → 이 경우 피벗 보정 옵션 필요(요청 시 추가)
  - **크기(Scale)**: 맵에 비해 크거나 작으면 `Scale Range` 또는 프리팹 Scale로 조정
    (실측 지도는 스케일이 클 수 있음)

---

## 3. 지형에 Collider 확인 (필수)

흩뿌리기는 **위에서 아래로 광선을 쏴서 지면을 찾습니다.**
따라서 육지(지형)에 Collider가 있어야 합니다.

- 메쉬 지형: 지형 선택 → `Add Component` → **Mesh Collider**
- Unity Terrain: 이미 콜라이더 있음 (추가 작업 없음)

---

## 4. 흩뿌리기 실행

1. Hierarchy 빈 곳 우클릭 → `Create Empty` (이름 예: `NatureScatter`)
2. 위치를 **맵 중앙**으로 이동
3. `Add Component` → **"Nature Scatter"**
4. **Prefabs** 배열 크기를 늘리고 나무·바위·덤불 프리팹을 드래그
5. **Area Size**를 맵 크기에 맞추기 (씬에 노란 사각형으로 범위 표시됨)
6. **Min Height**를 **수면 높이(Water Level)** 값으로 설정 → 바다엔 안 생기고 육지에만 깔림
7. 컴포넌트 우클릭(점 세 개 ⋮) → **"Scatter Now"**
8. 마음에 안 들면 우클릭 → **"Clear"** 후 값 조정해서 다시 실행

---

## 5. 설정값 설명

| 항목 | 설명 |
|------|------|
| **Prefabs** | 뿌릴 프리팹 목록 (나무/바위/풀, 받아둔 에셋도 가능) |
| **Count** | 배치할 개수 |
| **Area Size** | 뿌릴 정사각 범위 한 변 길이 (맵 크기에 맞추기) |
| **Raycast Height** | 위에서 아래로 쏘는 시작 높이 (지형보다 충분히 높게) |
| **Min Height** | 이 높이 아래엔 안 놓음 → 보통 수면 높이로 (바다 제외) |
| **Max Height** | 이 높이 위엔 안 놓음 → 산 정상 제외 등 |
| **Max Slope** | 이 경사(도)보다 가파르면 안 놓음 → 절벽 제외 |
| **Ground Mask** | 지형 레이어 (기본: 전부) |
| **Scale Range** | 크기를 무작위로 다양화 (예: 0.8 ~ 1.3) |
| **Random Yaw** | 무작위 회전 (자연스러움) |
| **Align To Slope** | 경사면 기울기에 맞춰 세우기 |

---

## 6. 자주 겪는 문제

- **하나도 안 생김** → 지형에 Collider 없음 / Min·Max Height 범위가 맞지 않음 / Area Size가 지형 밖
- **바다에도 생김** → Min Height를 수면 높이로 올리기
- **땅에 묻히거나 뜸** → 프리팹 피벗 문제 (피벗 보정 옵션 요청)
- **너무 작거나 큼** → Scale Range 또는 프리팹 Scale 조정
- **절벽에도 박힘** → Max Slope 낮추기 (예: 30)

---

## 7. 팁

- 받아둔 디테일 에셋을 메인으로, 로우폴리 모델을 보조로 섞으면 다양성이 좋아집니다.
- 종류별로 NatureScatter 오브젝트를 여러 개 두고(숲용, 바위용 등) 영역·밀도를 따로 관리해도 됩니다.
- 더 빽빽한 잔디는 별도 방식(빌보드/디테일)이 효율적입니다 — 필요 시 요청.
