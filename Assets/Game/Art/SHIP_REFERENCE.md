# 배 15종 — 모델링 레퍼런스

`Assets/Game/Data/Ships/` 에 등록된 ShipData 의 실제 값 + 시각적 특징 정리. Blender 작업 시 비주얼 가이드로 사용.

스탯 기호:

- ⚔ 대포 (cannonPower 1~30)
- ✈ 속도 (speed 1~10)
- 📦 화물 (cargoCapacity 10~1000)
- 🛡 내구도 (maxDurability 10~200)
- ⏱ 발사간격 (attackInterval 초)
- 💰 가격 (G)

---

## Tier 1 — 입문 / 소형 (800 ~ 1,500G)

작고 단순한 실루엣. 돛 1~2개. 모델링 부담 적음.

### 🇵🇹 라틴 카라벨 (CaravelaLatina) — 800G

- ⚔ 2 / ✈ 8 / 📦 50 / 🛡 50 / ⏱ 1.4
- **특징**: 작고 날씬, **삼각 돛(라틴 세일)** 2~3장
- **돛대 수**: 2~3개, 모두 라틴 세일
- **선체**: 길쭉하고 낮음. 갑판 단순
- **시대**: 15세기 초, 포르투갈 탐험 초기
- 참조: https://en.wikipedia.org/wiki/Caravel

### 🇸🇦 다우선 (Dhow) — 1,200G

- ⚔ 2 / ✈ 7 / 📦 80 / 🛡 55 / ⏱ 1.5
- **특징**: 큰 **삼각 돛(라틴 세일) 1장** 이 대부분. 우아한 곡선 선체
- **돛대 수**: 1~2개
- **선체**: 좁고 긴 hull, 뒷부분이 살짝 올라감
- **시대**: 중세~근세 인도양·아라비아
- 참조: https://en.wikipedia.org/wiki/Dhow

### 🛡 코그선 (Cog) — 1,500G

- ⚔ 3 / ✈ 3 / 📦 250 / 🛡 90 / ⏱ 2.0
- **특징**: **상자 모양** 묵직한 선체, **사각 돛 1장**
- **돛대 수**: 1개, 중앙
- **선체**: 둔하고 넓은 박스. 앞뒤로 높은 캐슬 (성벽 같은 구조물)
- **시대**: 중세 북유럽 한자동맹
- 참조: https://en.wikipedia.org/wiki/Cog_(ship)

## Tier 2 — 중급 / 무역 (2,500 ~ 4,000G)

본격 무역·전투 시작. 돛 2~3개, 다층 구조.

### 🇪🇺 캐릭선 (Carrack) — 2,500G

- ⚔ 6 / ✈ 4 / 📦 200 / 🛡 100 / ⏱ 1.5 (기본)
- **특징**: 거대한 박스형 선체, **사각 돛 + 라틴 돛 혼합**
- **돛대 수**: 3개 (전·중·후), 후방 돛대만 라틴
- **선체**: 앞뒤로 거대한 forecastle/sterncastle (높이 솟은 갑판)
- **시대**: 15~16세기, 대탐험 시대
- 참조: https://en.wikipedia.org/wiki/Carrack

### 🇮🇹 갤리선 (Galley) — 3,000G

- ⚔ 7 / ✈ 7 / 📦 60 / 🛡 70 / ⏱ 1.4
- **특징**: 길고 낮은 선체, **양옆에 노가 줄지어**. 1~2 돛대
- **돛대 수**: 1~2개, 작은 라틴 세일
- **선체**: 폭이 좁고 매우 길음. 노가 두드러진 시각 요소
- **시대**: 고대~중세 지중해
- 참조: https://en.wikipedia.org/wiki/Galley

### 🇨🇳 정크선 (Junk) — 3,500G

- ⚔ 5 / ✈ 5 / 📦 280 / 🛡 110 / ⏱ 1.7
- **특징**: **부채꼴 정크 세일** (대나무 살이 보이는 가로줄 돛), 평저 선체
- **돛대 수**: 2~4개, 모두 정크 세일
- **선체**: 평평한 바닥, 뒷부분 사각형
- **시대**: 중국 송~명대 (대형 보선 포함)
- 참조: https://en.wikipedia.org/wiki/Junk_(ship)

### 🇳🇱 플라이트선 (Fluyt) — 4,000G

- ⚔ 4 / ✈ 6 / 📦 350 / 🛡 80 / ⏱ 1.5 (기본)
- **특징**: 둥글둥글한 호리병 모양 선체, 사각 돛 위주
- **돛대 수**: 3개
- **선체**: 옆에서 보면 항아리 같음. 갑판이 좁고 화물칸 넓음
- **시대**: 17세기 네덜란드 황금기
- 참조: https://en.wikipedia.org/wiki/Fluyt

## Tier 3 — 대형 / 주력 (5,000 ~ 6,500G)

본격 함대. 3 돛대, 다층 갑판. 모델링 디테일 많음.

### 🇵🇹 카라벨 (Caravel) — 5,000G

- ⚔ 3 / ✈ 8 / 📦 60 / 🛡 50
- **특징**: 라틴 카라벨보다 큰 사이즈, **사각+라틴 혼합 돛**
- **돛대 수**: 3개 (앞은 사각, 중·후는 라틴)
- **선체**: 중형, 균형 잡힘. forecastle 작게
- 참조: https://en.wikipedia.org/wiki/Caravel

> 가격 5,000G — 기본값 600G 에서 사용자가 조정한 듯. 게임 디자인상 입문선이면 다시 600G 권장.

### 🇪🇸 산타마리아호 (SantaMaria) — 5,000G

- ⚔ 7 / ✈ 5 / 📦 180 / 🛡 110 / ⏱ 1.9
- **특징**: 카락의 한 종류. 콜럼버스의 대표선. 큰 사각 돛
- **돛대 수**: 3개
- **선체**: 캐릭선과 유사하나 약간 작고 둥글
- **시대**: 1492년, 콜럼버스 대서양 횡단
- 참조: https://en.wikipedia.org/wiki/Santa_Mar%C3%ADa_(ship)

### 🇪🇸 갈레온선 (Galleon) — 6,000G

- ⚔ 14 / ✈ 5 / 📦 180 / 🛡 150 / ⏱ 1.5 (기본)
- **특징**: 카락에서 발전한 전투형. **양옆에 대포구멍 줄지어**
- **돛대 수**: 3~4개, 사각 돛 + 후방 라틴
- **선체**: 길어진 hull, forecastle 낮아짐, sterncastle 여전히 높음
- **시대**: 16~17세기 스페인 무적함대
- 참조: https://en.wikipedia.org/wiki/Galleon

### 🇮🇹 갤리어스 (Galleass) — 6,500G

- ⚔ 12 / ✈ 6 / 📦 140 / 🛡 130 / ⏱ 1.6
- **특징**: 갤리 + 갈레온 융합. **노 + 큰 돛** 동시에. 대포 많음
- **돛대 수**: 3개, 사각 돛
- **선체**: 갤리보다 폭 넓고 높음, 노가 보임
- **시대**: 16세기 베네치아, 레판토 해전
- 참조: https://en.wikipedia.org/wiki/Galleass

### 🇰🇷 판옥선 (Panokseon) — 6,500G

- ⚔ 15 / ✈ 4 / 📦 100 / 🛡 160 / ⏱ 1.7
- **특징**: 평평한 바닥, **위에 판자로 만든 옥(屋)** 구조. 사각형
- **돛대 수**: 2개, 사각 돛
- **선체**: 평저선, 2층 갑판. 상부에 노 젓는 병사 보호용 지붕
- **시대**: 조선 중기 (16~17세기)
- 참조: https://ko.wikipedia.org/wiki/판옥선

## Tier 4 — 최상급 / 특수 (8,000G+)

가장 상징적인 배. 모델링 디테일과 시간 투자할 가치.

### 🇰🇷 거북선 (Geobukseon) — 8,000G

- ⚔ 18 / ✈ 5 / 📦 120 / 🛡 180 / ⏱ 1.5
- **특징**: **거북이 등딱지** 모양 덮개, 가시 박힌 등판. 용머리 뱃머리
- **돛대 수**: 2개 (접거나 펼침)
- **선체**: 판옥선 기반 + 철갑 덮개. 뱃머리에서 연기 / 불 뿜는 용머리
- **시대**: 임진왜란 (1592~1598), 이순신
- 참조: https://ko.wikipedia.org/wiki/거북선

### 🇺🇸 클리퍼 (Clipper) — 10,000G

- ⚔ 6 / ✈ 9 / 📦 110 / 🛡 90 / ⏱ 1.5
- **특징**: **매우 긴 hull, 거대한 돛** 여러 장. 날씬하고 우아
- **돛대 수**: 3~4개, 큰 사각 돛 + 보조 삼각 돛
- **선체**: 가장 길고 가장 좁은 비율. 뱃머리 뾰족
- **시대**: 19세기 중반 (시기상 우리 게임 시대보다 뒤지만 멋이 좋음)
- 참조: https://en.wikipedia.org/wiki/Clipper

### 🇬🇧 동인도무역선 (EastIndiaman) — 12,000G

- ⚔ 16 / ✈ 5 / 📦 500 / 🛡 170 / ⏱ 1.7
- **특징**: 거대한 갈레온, **대포구멍 양옆 가득**. 화물칸 어마어마
- **돛대 수**: 3~4개
- **선체**: 가장 크고 무거움. 갑판 다층, 후방 갤러리(창문 줄지어)
- **시대**: 17~19세기, 영국·네덜란드 동인도회사
- 참조: https://en.wikipedia.org/wiki/East_Indiaman

## 모델링 우선순위 추천

플레이어가 처음 마주칠 가능성이 높은 배 먼저:

1. **Caravel** — 시작 배. 가장 자주 보임
2. **Carrack** — 초중반 업그레이드
3. **Galleon** — 후반 주력 (해적·호위선이 자주 탐)
4. **Geobukseon** — 한국 상징, 마케팅용
5. **Fluyt** — 상선 NPC 가 많이 탐
6. **Junk** — 동아시아 NPC 시각 차별
7. **Panokseon** — 한국 NPC 색깔
8. **Galleass** — 지중해 NPC
9. **SantaMaria** — 카락과 유사하지만 별개 모델 원하면
10. 나머지는 게임 진척도와 시간 봐서

> 시간 제한 있으면 **5종만** 잘 만드는 게 15종 대충 만드는 것보다 좋음. Tier 1 은 절차적 큐브 배로 두고 Tier 2~4 만 모델로 가도 충분.

## 모델링 시 공통 체크

- ✅ 뱃머리(bow) = +Z 방향
- ✅ 수면(waterline) = Y 0
- ✅ 정규화 크기 약 1 유닛 (게임에서 자동 6배)
- ✅ < 2000 tri, 머티리얼 1~2개
- ✅ URP Lit 셰이더 호환
- ✅ 콜라이더 없음
